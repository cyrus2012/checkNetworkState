This repository is used to study how to download file from the internet in Android app (Kotlin)
It has divided into 5 parts and the explaination of the code is written on the blog.

1. Check network state and WiFi SSID
https://cygonna.blogspot.com/2024/01/android-check-network-state-and-wifi.html

2. Download file to public storage with DownloadManager(built-in library)
https://cygonna.blogspot.com/2024/01/android-download-file-to-public-storage.html

3. Download File into app specific storage with Retrofit
https://cygonna.blogspot.com/2024/01/download-file-into-app-specific-storage.html

4. Use okhttp to download file and show progress bar (it also can download to specific storage)
https://cygonna.blogspot.com/2024/02/use-okhttp-to-download-file-and-show.html

5. Unzipp file with Zip4j library
https://cygonna.blogspot.com/2024/02/unzipp-file-with-zip4j-library.html
